Privacy Policy

Last Updated: July 2026

1. Data Collection

Rye-Chan-Bot collects the minimum amount of data necessary to function. The data we may collect includes:

  • User IDs and Server IDs: To process commands and server-specific configurations.
  • Message Content: Only read temporarily to process commands (via Message Content Intent). Message content is not permanently stored or logged unless explicitly required for a specific command feature (like an anonymous confession board).

2. Data Usage

Collected data is strictly used to provide the bot's features, such as greeting new members, providing latency checks, or storing anonymous submissions for specific commands.

3. Data Storage and Security

Configuration and state data (such as confession counters) are stored locally on a secure server (Raspberry Pi 3B+) in a local database. We do not sell or share your data with any third parties.

4. Data Deletion

If you wish to have your server's data removed from the bot's database, you can remove the bot from your server. We routinely purge data for servers where the bot is no longer present.

5. Changes to this Policy

We may update this privacy policy from time to time. You will be notified of any significant changes via our support channels.
